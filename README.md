# BeatMoOod
BeatMoOod – A simple mood-based music recommendation web app built for Hack to the Beat 2026.
# BeatMood 🎵

## Inspiration
Music influences emotions deeply. I wanted to create a simple retro-themed mood-based music recommendation system.

## What it does
Users select their mood (Happy, Sad, Energetic, Relaxed), and the system suggests songs accordingly.

## How I built it
Built using HTML, CSS, and JavaScript with simple condition-based logic.

## Challenges
Due to limited time (1 hour), I focused on building a clean working prototype.

## What I learned
I learned how to quickly convert an idea into a working web prototype under time pressure.

## Future Scope
- Connect with Spotify API
- Add AI-based mood detection
- Improve UI with retro theme design
